### 1. 배열

10개의 int형 정보를 담을 수 있는 array라는 배열. (0부터 시작)
```c# 
int [] array = new int[10];
```


배열 원소를 지정해 줄 수도 있다.  
```c# 
int [] num = { 1,3,5,7 };
```
> num.Length ? 배열 길이(원소의 갯수)


### 2. for문 (반복문)
기본형식
```c#
for( 1. 초기값 ; 2. 실행조건문 ; 4. 반복 후 실행 )  
{  
   3. 반복문 내용   
}
```
응용
```c#
for(int i=0; i<num.Length ;i++)
{
 Debug.Log(num[i].name); //배열 원소의 이름 콘솔에 찍어낸다.
}
```
